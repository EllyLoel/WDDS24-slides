---
title: '`::view-transition-group(*)`'
visuallyHiddenTitle: true
notes: |
  - Default animation duration for view transitions is 1/4 second.
---

```css
:root::view-transition-group(*) {
	position: absolute;
	top: 0;
	left: 0;

	animation-duration: 0.25s;
	animation-fill-mode: both;
}
```
